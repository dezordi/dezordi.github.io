Tools and Repositories

NCBI-seq retriever

Python toolbox to automatic recovery information of NCBI.

At the moment, this python script uses the Bio.Entrez module to automatic recovery NCBI information. There are 4 principal functions:

  • • Recovery genbank information from nucleotide sequences;
  • • Recovery cds translated to amino acids from nucleotide sequences;
  • • Recovery nucleotide or aminoacid sequences;
  • • Recovery taxonomy information of ncbi acess IDs.
For more information, Click Here!

Mosquito Taxonomy

General information about mosquitoes, specially about taxonomy

Several information about Culicidae taxonomy based on Mosquito Taxonomic Inventory and:

  • • Links for specific databases;
  • • Plots based on most updated culicdae taxonomy;
  • • Script to create plots about specific genus;
For more information, Click Here!

SARS-CoV-2 Tools

Scripts developed for SARS-CoV-2 research.

A set of python scripts to work with SARS-CoV-2 genomes based on pattern names of GISAID, to:

  • • Clusterize genomes by country;
  • • Mask alignment sites;
  • • Clean fasta file by length and N region percentage;
  • • Sample fasta by Day or Week of the collection date;
  • • Automatize iTOL files construction;
  • • Get SNP diversity by genomic region;
  • • Get nucleotide or aminoacid diversity by alignment sites;
For more information, Click Here!

SARS-CoV-2 Genome Assembly

Workflow developed to assembly SARS-CoV-2 genome from illumina paired-end data

A shell script and a docker container to:

  • • Quality controll of fastq data;
  • • Genome assembly guided by reference;
  • • SARS-CoV-2 lineage analysis;
  • • SARS-CoV-2 mutation analysis;
  • • Minor variant analysis;
  • • Analysis of assembly metrics;
